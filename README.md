# Projekt: Dungeon Crawler

## Tech stack

- .NET SDK 8.0
- Godot Editor 4.6.1

## Klonowanie projektu

```bash
git clone git@github.com:orlowski-dev/RPI.git &&
cd RPI &&
dotnet build
```

> [!NOTE]
> Jeśli są warningi o Invalid resource UID to neleży w Godot Projekt > Narzędzia > Uaktualnij pliki projektu...
